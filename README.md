# Syro-Malabar Qurbana

A bilingual (English & Malayalam) Progressive Web App (PWA) for the Holy Qurbana â€” Syro-Malabar Rite.

## Live App

[https://professorgeorge.github.io/qurbana/](https://professorgeorge.github.io/qurbana/)

## Features

- Bilingual liturgical text (English & Malayalam)
- Installable as a PWA (works offline)
- Auto-updating service worker
- Google Fonts (Cinzel, Inter, Noto Sans Malayalam)
- Section-by-section navigation with progress tracking

## Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [vite-plugin-pwa](https://vite-pwa-org.netlify.app/)
- Deployed via [GitHub Actions](https://github.com/features/actions) to [GitHub Pages](https://pages.github.com/)
